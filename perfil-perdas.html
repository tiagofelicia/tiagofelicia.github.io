Cálculo de Perdas

Para calcular as Perdas em Baixa Tensão (BT) é necessário aplicar a seguinte fórmula:

(1+pAT/RT) x (1+pAT) x (1+pMT) x (1+pBT)

Para calcular as Perdas em Média Tensão (MT) é necessário aplicar a seguinte fórmula:

(1+pAT/RT) x (1+pAT) x (1+pMT)
