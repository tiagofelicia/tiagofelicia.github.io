Sobre este mapa

O que mostra este mapa?

Este mapa apresenta a percentagem de energia renovável no mix de produção de eletricidade de cada país europeu.

As cores variam do verde (alta percentagem renovável) ao castanho (baixa percentagem renovável). A escala de luminosidade foi escolhida para melhor acessibilidade a utilizadores daltónicos.

Como é calculada a percentagem renovável?

A percentagem renovável é calculada como: (Solar + Eólica + Hídrica + Biomassa + Outras Renováveis) / Produção Total × 100.

Os valores de produção são totais diários em GWh, obtidos a partir dos dados de produção pública líquida da API Energy-Charts.

Bombagem hidráulica (hydro pumped storage) é, por default, excluída tanto do numerador como do denominador — é a convenção adotada pela energy-charts.info. Podes alterar este comportamento no toggle "Excluir bombagem / Incluir bombagem".

O que faz o toggle "Excluir / Incluir bombagem"?

A hidráulica de bombagem não é geração primária — é energia armazenada previamente (bombeada de uma albufeira inferior para uma superior durante horas de excedente) e re-turbinada mais tarde. A origem dessa energia pode ter sido fóssil ou renovável, não é intrinsecamente verde. Por isso, nesta página, a bombagem NUNCA é contabilizada como renovável — independentemente do toggle.

O toggle controla apenas o denominador da percentagem renovável e das percentagens por coluna:

  • Excluir bombagem (default, alinhado com energy-charts.info): bombagem fica fora do denominador. A soma das colunas (exceto bombagem) atinge os 100% da produção não-bombeada.
  • Incluir bombagem: bombagem entra no denominador como "mais uma origem". A soma de todas as colunas (incluindo bombagem) é 100%.

Na tabela, a bombagem aparece sempre numa coluna autónoma (entre "Out. Renov." e "Fóssil"), para ser visível e ordenável independentemente do toggle.

% sobre Produção vs. % sobre Consumo — qual a diferença?

O seletor "% sobre Produção" / "% sobre Consumo" (junto aos seletores de data) permite escolher o denominador da percentagem renovável:

  • % sobre Produção (default): renováveis / produção total do país. Reflete o peso das renováveis no mix nacional gerado.
  • % sobre Consumo: renováveis / consumo (load) do país. Reflete o peso das renováveis face à procura. É a métrica usada pela energy-charts.info.

Um país exportador com muita renovável pode ter % sobre Consumo > 100% (produz mais renovável do que consome). Um país importador pode ter % sobre Consumo inferior à % sobre Produção.

A opção escolhida aplica-se ao mapa, gráfico de evolução e tabela de produção por país, e é incluída no link de partilha.

Porque é que todas as zonas de um país têm a mesma cor?

A API Energy-Charts devolve dados de produção ao nível do país, não por zona de licitação. Por isso, países como a Noruega (5 zonas), Suécia (4 zonas) ou Itália (6+ zonas) partilham a mesma cor em todas as suas zonas.

Os preços de eletricidade, por outro lado, variam entre zonas — consulte o Mapa de Preços para essa informação.

Que fontes de energia são consideradas renováveis?

São consideradas renováveis as seguintes fontes:

  • Solar (fotovoltaica)
  • Eólica (wind onshore e wind offshore)
  • Hídrica (fio-de-água, albufeira e bombagem — apenas geração)
  • Biomassa (inclui Biomassa florestal, biogás e a fração biodegradável dos resíduos sólidos urbanos)
  • Outras Renováveis (inclui geotermia)

As fontes fósseis (gás natural, carvão, petróleo) e nuclear não são consideradas renováveis.

De onde vêm os dados?

Os dados são obtidos através da API Energy-Charts do Fraunhofer ISE, que agrega dados de produção de eletricidade de 28 países europeus.

Os dados são atualizados diariamente de forma automática.

Porque é que alguns países não têm dados?

Zonas a cinzento indicam ausência de dados para o dia selecionado. Isto pode acontecer porque:

  • Os dados ainda não foram publicados
  • O país não está disponível na API Energy-Charts
  • Houve uma falha temporária na recolha de dados
