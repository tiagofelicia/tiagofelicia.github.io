Compreender os Preços do OMIE Diário

O que é o OMIE e que preços são mostrados aqui?

O OMIE (Operador do Mercado Ibérico de Energia) gere o mercado grossista diário e intradiário de eletricidade para Portugal e Espanha (MIBEL). Os preços resultam de leilões onde a energia é comprada e vendida para o dia seguinte.

Esta página mostra esses preços (€/MWh) para cada intervalo de 15 minutos (quarto-horário), já corrigidos para a hora de Portugal Continental (PT). Também são apresentados os preços originais para Espanha (ES) para comparação.

O que significam as médias Simples, Bi-Horária e Tri-Horária?

São médias dos preços quarto-horários de Portugal (€/MWh) durante diferentes períodos do dia selecionado, calculadas com base nos ciclos standard definidos pela ERSE:

  • Simples: Média de todos os preços *reais* do OMIE no dia selecionado (ver nota abaixo sobre o último dia).
  • Bi-Horário (Diário/Semanal): Média dos preços durante os períodos de Vazio e Fora de Vazio.
  • Tri-Horário (Diário/Semanal): Média dos preços durante os períodos de Ponta, Cheias e Vazio.

Pode consultar as horas dos períodos em https://www.tiagofelicia.pt/periodos-horarios.html

Nota Importante: No dia mais recente apresentado, os cálculos de médias, máximo e mínimo para Portugal excluem os últimos quartos de hora (normalmente após as 23h PT), pois os mesmos ainda não são conhecidos.

Como é calculada a média de Espanha?

A média, máximo e mínimo para Espanha apresentados referem-se às 24 horas do dia espanhol. Como o dia espanhol começa e acaba uma hora mais cedo que o português, o cálculo inclui:

  • Os preços de Espanha da última hora (23:00-24:00) na hora de Portugal do dia anterior.
  • Os preços de Espanha das 23 horas (00:00-23:00) na hora de Portugal do dia selecionado.
O que são os Futuros OMIP?

OMIP é o mercado onde se negoceiam os preços de eletricidade para o futuro (próximo dia, mês, trimestre, ano). Os valores apresentados representam a expectativa do mercado para os preços futuros.

O que significa o "Futuro do dia"?

O valor "Futuro do dia" apresentado nos cartões de Portugal e Espanha é o preço do contrato futuro diário negociado no OMIP para o dia específico que está a visualizar. Este valor representa a expectativa que o mercado tinha (na data de atualização OMIP indicada) sobre qual seria o preço médio nesse dia.

É especialmente útil para comparar a expectativa (futuro) com a realidade (média OMIE calculada), particularmente no dia mais recente.

Os "Futuros Diários OMIP (PT)" são diferentes?

Sim. O cartão "Futuros Diários OMIP (PT)" mostra os preços dos contratos futuros diários para os próximos dias (até um máximo de 3), tal como negociados no OMIP. Servem para dar uma indicação das expectativas do mercado para o futuro próximo, para além do dia selecionado.

Estes preços OMIE são os que pago na fatura?

Não diretamente, a menos que tenha uma tarifa indexada ao OMIE.

  • Se tem uma tarifa indexada (ou dinâmica), o preço que paga pela energia varia hora a hora (ou quarto de hora) e está diretamente ligado a estes valores do OMIE, acrescido de uma margem do comercializador e outras taxas.
  • Se tem uma tarifa fixa, o preço por kWh é constante, definido pelo seu comercializador. No entanto, os preços do OMIE influenciam o custo de aquisição de energia para o comercializador e, indiretamente, os preços das tarifas fixas que ele oferece.

Lembre-se que a sua fatura final inclui sempre outras componentes como tarifas de acesso às redes, taxas e impostos.

De onde vêm os dados e qual o período disponível?

Os dados são processados através da recolha automática de informação pública dos websites do OMIE (para os preços diários "reais") e do OMIP (para os futuros).
