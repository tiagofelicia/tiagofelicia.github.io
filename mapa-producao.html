Sobre este mapa

O que mostra este mapa?

Este mapa apresenta a percentagem de energia renovável no mix de produção de eletricidade de cada país europeu.

As cores variam do verde (alta percentagem renovável) ao castanho (baixa percentagem renovável). A escala de luminosidade foi escolhida para melhor acessibilidade a utilizadores daltónicos.

Como é calculada a percentagem renovável?

A percentagem renovável é calculada como: (Solar + Eólica + Hídrica + Biomassa) / Produção Total × 100.

Os valores de produção são totais diários em GWh, obtidos a partir dos dados de produção pública líquida da API Energy-Charts.

Porque é que todas as zonas de um país têm a mesma cor?

A API Energy-Charts devolve dados de produção ao nível do país, não por zona de licitação. Por isso, países como a Noruega (5 zonas), Suécia (4 zonas) ou Itália (6+ zonas) partilham a mesma cor em todas as suas zonas.

Os preços de eletricidade, por outro lado, variam entre zonas — consulte o Mapa de Preços para essa informação.

Que fontes de energia são consideradas renováveis?

São consideradas renováveis as seguintes fontes:

  • Solar (fotovoltaica)
  • Eólica onshore (wind onshore)
  • Eólica offshore (wind offshore)
  • Hídrica (fio-de-água, albufeira e bombagem — apenas geração)
  • Biomassa (inclui geotermia e resíduos)

As fontes fósseis (gás natural, carvão, petróleo) e nuclear não são consideradas renováveis.

De onde vêm os dados?

Os dados são obtidos através da API Energy-Charts do Fraunhofer ISE, que agrega dados de produção de eletricidade de 28 países europeus.

Os dados são atualizados diariamente de forma automática.

Porque é que alguns países não têm dados?

Zonas a cinzento indicam ausência de dados para o dia selecionado. Isto pode acontecer porque:

  • Os dados ainda não foram publicados
  • O país não está disponível na API Energy-Charts
  • Houve uma falha temporária na recolha de dados
